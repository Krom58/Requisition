# ระบบเบิกของ (Requisition Management System)

**Requisition Management System** คือโปรแกรมสำหรับ **จัดการการเบิกสินค้า / วัสดุ** (Request / Requisition) ภายในองค์กร เช่น การสร้างรายการเบิก การตรวจสอบรายการ และการบันทึกข้อมูลลงฐานข้อมูล  

ระบบช่วยให้การติดตามการใช้ทรัพยากรภายในองค์กรมีความเป็นระเบียบ ลดความผิดพลาดในการบันทึกข้อมูล และสามารถตรวจสอบประวัติการเบิกย้อนหลังได้ง่าย

---

## ความเป็นมา (Background)

ในหลายองค์กร การเบิกวัสดุหรือสินค้า เช่น  

- อุปกรณ์สำนักงาน  
- วัตถุดิบ  
- สินค้าคงคลัง  
- อุปกรณ์ IT  

มักใช้วิธีบันทึกแบบ manual เช่น  

- เขียนลงกระดาษ  
- บันทึกใน Excel  
- แจ้งผ่าน chat  

ซึ่งอาจทำให้เกิดปัญหา เช่น  

- ข้อมูลสูญหาย  
- ตรวจสอบย้อนหลังยาก  
- ไม่ทราบจำนวนคงเหลือจริง  
- เกิดการเบิกซ้ำโดยไม่จำเป็น  

ดังนั้น **Requisition Management System** จึงถูกพัฒนาขึ้นเพื่อช่วยให้สามารถจัดการรายการเบิกได้ในระบบเดียว โดยข้อมูลทั้งหมดจะถูกเก็บในฐานข้อมูลอย่างเป็นระบบ ทำให้สามารถค้นหา ตรวจสอบ และวิเคราะห์ข้อมูลได้สะดวกมากขึ้น  

### วัตถุประสงค์ของระบบ

1. จัดการข้อมูลการเบิกสินค้าอย่างเป็นระบบ  
2. ลดความผิดพลาดในการบันทึกข้อมูล  
3. สามารถตรวจสอบประวัติการเบิกย้อนหลังได้  
4. ช่วยติดตามจำนวนการใช้งานสินค้า  

---

## ภาพรวมระบบ (Overview)

ระบบช่วยจัดการกระบวนการเบิกสินค้า เช่น

- สร้างรายการเบิกสินค้า
- บันทึกจำนวนสินค้า
- ตรวจสอบข้อมูลรายการเบิก
- ค้นหาประวัติการเบิกย้อนหลัง
- จัดเก็บข้อมูลในฐานข้อมูล

---

## คุณสมบัติหลัก (Key Features)

### Requisition Management
- เพิ่ม / แก้ไข / ลบ รายการเบิก
- ระบุจำนวนสินค้า

### Record Tracking
- บันทึกข้อมูลการเบิก
- ตรวจสอบประวัติย้อนหลัง

### Search Function
- ค้นหารายการเบิก
- ดูข้อมูลย้อนหลังได้ง่าย

### Database Integration
- จัดเก็บข้อมูลในฐานข้อมูล
- ลดความซ้ำซ้อนของข้อมูล

### User Interface
- ใช้งานง่าย
- เหมาะสำหรับใช้ภายในองค์กร
